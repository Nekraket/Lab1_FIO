# Lab1_FIO

## О проекте

Программа для вычисления вида треугольника и координат трех его вершин для отрисовки в поле 100×100 px.

## Функциональность

- Проверка валидности входных данных (вещественные положительные числа)
- Определение типа треугольника:
  - Равносторонний
  - Равнобедренный
  - Разносторонний
  - Не треугольник
- Вычисление координат вершин (X, Y) с масштабированием под область 100×100 px
- Логирование всех запросов (успешных и неуспешных) в файл и консоль

## Входные данные

Три строки, представляющие длины сторон A, B, C (вещественные числа типа float).

## Выходные данные

- Тип треугольника
- Массив из трех координат вершин (int, int)
